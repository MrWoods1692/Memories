📋 架构概述

Memories 是一个运行在 Android 设备上的图片分享与管理平台，内嵌两个基于 NanoHTTPD 的 HTTP 服务器。

🖥️

API 服务器

端口 8080，处理所有 REST API 请求

📦

管理面板

端口 8081，提供 React 管理界面静态文件

🔒

局域网安全

只接受局域网请求，自动 CORS 支持

🗄️

SQLite 存储

4 张表：images、users、config、banned_users

基础 URL

http://<设备局域网IP>:8080

通用约定

  • POST 请求使用 application/x-www-form-urlencoded 编码
  • 响应为 application/json 或 text/plain
  • 所有响应自动添加 CORS 头，允许跨端口调用
  • 局域网用户自动获得管理员权限

🔐 认证与授权

角色体系

角色值名称权限
0未注册用户仅可上传图片
1审核员 (Reviewer)审核图片（通过/拒绝）
2管理员 (Admin)审核图片 + 管理用户 + 封禁管理 + 修改配置

认证方式

  • 局域网自动授权：来自局域网的请求自动获得管理员权限
  • Header 认证：通过 x-user-qq 请求头传递 QQ 号，后端查用户表确定角色

权限矩阵
